# SSITAO Code Platform 实施计划

## 概述

本文档详细描述权限管理体系和低代码功能的实施步骤。

## 任务分解

### 阶段一：基础设施搭建 (第1周)

#### 1.1 多租户模块实现

**任务1.1.1: 租户实体与DTO创建**
- 创建 `IamTenant` 实体类
- 创建 `IamTenantDTO` 数据传输对象
- 创建租户Command类(Create/Update)
- 路径: `ssitao-code-modulars/ssitao-code-modular-iam/.../system/domain/model/IamTenant.java`

**任务1.1.2: 租户Repository与Service**
- 创建 `IamTenantRepository` 接口
- 创建 `IamTenantAppService` 服务
- 实现基本的CRUD操作

**任务1.1.3: 租户管理控制器**
- 创建 `IamTenantController`
- 实现租户的增删改查API

**任务1.1.4: 多租户拦截器**
- 创建 `TenantInterceptor` 租户拦截器
- 实现租户ID解析和注入
- 配置拦截器注册

**任务1.1.5: 租户数据隔离**
- 创建租户SQL过滤器
- 修改BaseMapper支持租户过滤
- 配置租户上下文

#### 1.2 组织机构管理(五级结构)

**任务1.2.1: 组织机构模型完善**
- 完善 `IamGroup` 集团实体
- 完善 `IamCompany` 公司实体
- 完善 `IamDepartment` 部门实体
- 完善 `IamPost` 岗位实体

**任务1.2.2: 用户组织关联**
- 创建 `IamUserOrg` 用户组织关联
- 实现用户多组织支持
- 实现主组织设置

**任务1.2.3: 组织机构API**
- 集团/公司/部门/岗位 CRUD API
- 组织树形结构API
- 用户分配组织API

## 技术要点

### 多租户实现
- 使用AOP拦截器实现租户上下文
- ThreadLocal存储当前租户
- MyBatis拦截器注入租户条件

### 动态API
- 使用Java Proxy动态代理
- MyBatis-Flex动态SQL
- SpEL表达式解析

### 性能优化
- 实体配置缓存
- 字段定义本地缓存
- 动态SQL预编译
