功能说明

动态CRUD允许您动态创建和管理数据实体，无需编写代码。

请通过 实体管理 页面创建实体，然后使用动态接口进行数据操作。

前往实体管理

可用接口

操作 URL 说明
GET /meta/dynamic/data/{entityCode} 查询数据
POST /meta/dynamic/data/{entityCode} 新增数据
PUT /meta/dynamic/data/{entityCode}/{id} 更新数据
DELETE /meta/dynamic/data/{entityCode}/{id} 删除数据
